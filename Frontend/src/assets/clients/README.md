# Client logos

Drop your client **logo** files here and they appear automatically in the
"Trusted by" marquee on the homepage — no code changes needed.

- Formats: `.png` `.svg` `.webp` `.jpg`
- Best results: **transparent background**, white or single-color logos
  (they're shown grayscale and light up to full color on hover).
- Name the files after the company (used as the image alt text), e.g.
  `acme.png`, `traffic.svg`. A leading number controls order: `01-acme.png`.

The section hides itself when this folder is empty, so nothing broken ships.
