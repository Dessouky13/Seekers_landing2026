## Phase 1 — Get found by search engines (Week 1) ⭐ highest priority

### 1.1 Google Search Console (do this first)
1. Go to **https://search.google.com/search-console** and sign in with the company Google account.
2. Add a property → choose **URL prefix** → enter `https://www.seekersai.org`.
3. Verify ownership. Easiest non‑coding option: **Domain/DNS (TXT record)** — Search Console gives you a TXT value; add it in the domain's DNS (Namecheap/registrar). *(Alternatively, send the "HTML tag" token to the developer 🔧 — they'll paste it in 2 minutes.)*
4. After verifying: **Sitemaps** → submit `sitemap.xml`.
5. **URL Inspection** → paste each page URL → **Request indexing** for: `/`, `/solutions`, `/industries`, `/about`, `/faq`, `/contact`.

**Report to me:** ✅ verified (yes/no), ✅ sitemap submitted, and how many pages show as **Indexed** (screenshot of the Coverage/Pages report).

### 1.2 Bing Webmaster Tools
1. Go to **https://www.bing.com/webmasters** → sign in → **Import from Google Search Console** (fastest) or add the site manually.
2. Submit the sitemap: `https://www.seekersai.org/sitemap.xml`.

**Report to me:** ✅ added + sitemap submitted. *(Bing also powers ChatGPT's web search — this matters for GEO.)*
